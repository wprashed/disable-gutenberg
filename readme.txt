Disable Gutenberg editor for selected post types.

== Description ==

Disable Gutenberg allows you to disable the block editor (Gutenberg) for specific post types while keeping it active for others.

Features:
* Select post types to disable Gutenberg
* Excludes Media (attachments) automatically
* Top-level menu in admin for easy access
* Works with any WordPress theme

== Installation ==

1. Upload `disable-gutenberg` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to **Disable Gutenberg** in the admin menu.
4. Check the post types where Gutenberg should be disabled and click **Save Changes**.

== Frequently Asked Questions ==

= Can I disable Gutenberg for all post types? =
Currently, you can select post types to disable Gutenberg. Any unchecked post types will continue using Gutenberg.

= Can I re-enable Gutenberg for a post type later? =
Yes, simply uncheck the post type and save changes.
