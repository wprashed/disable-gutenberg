Enable or disable Gutenberg editor for specific post types in WordPress.

== Description ==

Disable Gutenberg lets you select which post types should use the Classic Editor instead of Gutenberg.  
- Easy-to-use admin page  
- Select multiple post types  
- Excludes Media (attachment) post type  
- Works with any WordPress theme  

== Installation ==

1. Upload `disable-gutenberg` to the `/wp-content/plugins/` directory.  
2. Activate the plugin through the **Plugins** menu in WordPress.  
3. Navigate to **Disable Gutenberg** menu in admin.  
4. Check the post types where Gutenberg should be disabled and click **Save Changes**.

== Frequently Asked Questions ==

= Can I disable Gutenberg for all post types? =
Currently, you can disable Gutenberg per post type. All unchecked post types will continue using Gutenberg.

= Can I re-enable Gutenberg for a post type later? =
Yes, simply uncheck the post type and save changes.
